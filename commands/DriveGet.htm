DriveGet

Ruft verschiedene Arten von Informationen über die Laufwerke des Computers ab.

DriveGet, AusgabeVar, Unterbefehl , Wert

Der AusgabeVar-Parameter ist Name der Variable, in der das Ergebnis gespeichert werden soll. Die Parameter Unterbefehl und Wert sind voneinander abhängig und können wie folgt genutzt werden.

Unterbefehle

Geben Sie für Unterbefehl Folgendes an:

  • List: Ermittelt eine Zeichenkette aus Buchstaben, die jeweils einen Laufwerksbuchstaben im System repräsentieren.
  • Capacity: Ermittelt die Gesamtkapazität eines bestimmten Pfads in Megabyte.
  • FileSystem: Ermittelt den Dateisystemtyp eines bestimmten Laufwerks.
  • Label: Ermittelt die Datenträgerbezeichnung eines bestimmten Laufwerks.
  • Serial: Ermittelt die Datenträgerseriennummer eines bestimmten Laufwerks.
  • Type: Ermittelt den Laufwerkstyp eines bestimmten Pfads.
  • Status: Ermittelt den Status eines bestimmten Pfads.
  • StatusCD: Ermittelt den Medienstatus eines CD- oder DVD-Laufwerks.

List

Ermittelt eine Zeichenkette aus Buchstaben, die jeweils einen Laufwerksbuchstaben im System repräsentieren. Zum Beispiel: ACDEZ.

DriveGet, AusgabeVar, List , Typ

Lässt man Typ weg, werden alle Laufwerkstypen abgerufen. Geben Sie ansonsten für Typ eines der folgenden Wörter an, um nur einen bestimmten Laufwerkstyp abzurufen: CDROM, REMOVABLE, FIXED, NETWORK, RAMDISK, UNKNOWN.

Capacity

Ermittelt die Gesamtkapazität von Pfad (z. B. C:\) in Megabyte.

DriveGet, AusgabeVar, Capacity, Pfad

Mit DriveSpaceFree kann der freie Speicherplatz abgerufen werden. Es ist möglich, das Wort Cap anstelle von Capacity zu verwenden.

FileSystem

Ermittelt den Dateisystemtyp von Laufwerk.

DriveGet, AusgabeVar, FileSystem, Laufwerk

Laufwerk ist der Laufwerksbuchstabe, gefolgt von einem Doppelpunkt und einem optionalen umgekehrten Schrägstrich, oder ein UNC-Name wie \\server1\share1. AusgabeVar wird auf eines der folgenden Wörter gesetzt: FAT, FAT32, NTFS, CDFS (typisch für CDs), UDF (typisch für DVDs). AusgabeVar wird leer gemacht und ErrorLevel auf 1 gesetzt, wenn das Laufwerk kein formatiertes Medium enthält. Es ist möglich, das Wort FS anstelle von FileSystem zu verwenden.

Label

Ermittelt die Datenträgerbezeichnung von Laufwerk.

DriveGet, AusgabeVar, Label, Laufwerk

Laufwerk ist der Laufwerksbuchstabe, gefolgt von einem Doppelpunkt und einem optionalen umgekehrten Schrägstrich, oder ein UNC-Name wie \\server1\share1. Die Bezeichnung kann wie folgt geändert werden: Drive, Label, C:, MeineBezeichnung.

Serial

Ermittelt die Datenträgerseriennummer von Laufwerk als dezimalen Integer.

DriveGet, AusgabeVar, Serial, Laufwerk

Laufwerk ist der Laufwerksbuchstabe, gefolgt von einem Doppelpunkt und einem optionalen umgekehrten Schrägstrich, oder ein UNC-Name wie \\server1\share1. Siehe Format() oder SetFormat, wie man diesen Wert in einen hexadezimalen Wert umwandeln kann.

Type

Ermittelt den Laufwerkstyp von Pfad.

DriveGet, AusgabeVar, Type, Pfad

AusgabeVar wird auf eines der folgenden Wörter gesetzt: Unknown, Removable, Fixed, Network, CDROM, RAMDisk.

Status

Ermittelt den Status von Pfad.

DriveGet, AusgabeVar, Status, Pfad

AusgabeVar wird auf eines der folgenden Wörter gesetzt: Unknown (könnte auf unformatiert/RAW hinweisen), Ready, NotReady (typisch für leere Wechseldatenträger), Invalid (Pfad existiert nicht oder ist ein Netzwerklaufwerk, auf das momentan nicht zugegriffen werden kann, usw.)

StatusCD

Ermittelt den Medienstatus eines CD- oder DVD-Laufwerks.

DriveGet, AusgabeVar, StatusCD , Laufwerk

Laufwerk ist der Laufwerksbuchstabe, gefolgt von einem Doppelpunkt. Lässt man Laufwerk weg, wird das Standard-CD/DVD-Laufwerk verwendet. AusgabeVar wird leer gemacht, wenn der Status nicht abgerufen werden kann. Ansonsten wird es auf eine der folgenden Zeichenketten gesetzt:

Zkette Beschreibung
not ready Das Laufwerk ist nicht bereit für den Zugriff, zum Beispiel aufgrund eines Schreibvorgangs. Bekannte Einschränkung: "not ready" tritt auch auf, wenn das Laufwerk eine DVD statt CD enthält.
open Das Laufwerk enthält keine CD, oder ist ausgefahren.
playing Das Laufwerk spielt gerade eine CD ab.
paused Die zuvor abgespielte Audio- oder Videowiedergabe ist nun pausiert.
seeking Das Laufwerk ist am Suchen.
stopped Das Laufwerk enthält eine CD, aber greift momentan nicht darauf zu.

Dieser Unterbefehl wird eventuell nicht funktionieren, wenn das angegebene Laufwerk ein Netzwerk- oder Nicht-CD/DVD-Laufwerk ist; wenn dieser Befehl in solchen Fällen oder aus einem anderen Grund fehlschlägt, wird AusgabeVar leer gemacht und ErrorLevel auf 1 gesetzt.

Wenn das Laufwerk vor Kurzem geschlossen wurde, kann es zu einer Verzögerung kommen, bevor der Unterbefehl zu Ende ist.

Mit dem Drive-Befehl kann ein Laufwerk ein- oder ausgefahren werden.

Fehlerbehandlung

[v1.1.04+]: Dieser Befehl ist in der Lage, bei Misserfolg eine Ausnahme auszulösen. Weitere Informationen finden Sie unter Laufzeitfehler.

ErrorLevel wird bei Misserfolg auf 1 und bei Erfolg auf 0 gesetzt.

Bemerkungen

Einige dieser Unterbefehle akzeptieren als Pfad oder Laufwerk auch Netzwerkfreigabenamen wie \\MeinServer\MeinFreigabe\.
