Sprachcodes

Die folgende Liste enthält Sprachnamen, die jeweils einem Sprachcode (auch bekannt als Locale Identifier, kurz LCID) zugeordnet sind, der in der A_Language-Variable enthalten sein kann. Der Sprachcode selbst sind die letzten vier Ziffern links vom Gleichheitszeichen. Wenn A_Language z.B. 0436 enthält, ist Afrikaans (Südafrika) die Standardsprache des Systems.

Hinweis: Die Buchstaben eines Codes können entweder groß oder klein geschrieben sein.

Sie können A_Language direkt mit einem der unteren 4-stelligen Codes vergleichen, z.B. if (A_Language = "0436"). Alternativ können Sie die gesamte Liste in ein Skript einfügen und dann auf den Namen der aktuellen Sprache zugreifen, wie am Ende der Liste gezeigt.

Das folgende Skript wurde zum Erstellen der obigen Liste verwendet (erfordert Windows Vista oder höher):

Gui, Add, ListView, w500 r20, LCID|Display Name|Locale Name
Gui, Show

LOCALE_ALLOW_NEUTRAL_NAMES := 0x08000000
LOCALE_SENGLISHDISPLAYNAME := 0x72
LOCALE_SLOCALIZEDDISPLAYNAME := 0x2
loop 0xFFFF
{
    LCID := Format("{:04X}", A_Index)
    if (LCID ~= "(04|08|0C|14|20|24|28|2C|30|34|38|3C|40|44|48|4C)00")
        continue  ; Standard- und transiente LCIDs überspringen.
    LocaleName := LCIDToLocaleName(A_Index, LOCALE_ALLOW_NEUTRAL_NAMES)
    if not LocaleName
        continue  ; Unbekannte LCIDs überspringen.
    DisplayName := GetLocaleInfo(LocaleName, LOCALE_SENGLISHDISPLAYNAME)
    ; DisplayName := GetLocaleInfo(LocaleName, LOCALE_SLOCALIZEDDISPLAYNAME)
    LV_Add(, LCID, DisplayName, LocaleName)
    ; Clipboard .= LCID "`t" DisplayName "`t" LocaleName "`n"
}
LV_ModifyCol()  ; Breite jeder Spalte an ihren Inhalt anpassen.

LCIDToLocaleName(LCID, Flags := 0)
{
    reqBufSize := DllCall("LCIDToLocaleName", "UInt", LCID, "Ptr", 0, "UInt", 0, "UInt", Flags)
    size := VarSetCapacity(out, reqBufSize)
    DllCall("LCIDToLocaleName", "UInt", LCID, "Ptr", &out, "UInt", size, "UInt", Flags)
    return StrGet(&out, "UTF-16")
}

GetLocaleInfo(LocaleName, LCType)
{
    reqBufSize := DllCall("GetLocaleInfoEx", "WStr", LocaleName, "UInt", LCType, "Ptr", 0, "UInt", 0)
    size := VarSetCapacity(out, reqBufSize)
    DllCall("GetLocaleInfoEx", "WStr", LocaleName, "UInt", LCType, "Ptr", &out, "UInt", size)
    return StrGet(&out, "UTF-16")
}
