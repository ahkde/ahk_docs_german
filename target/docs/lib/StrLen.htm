StrLen()

Ruft die Anzahl der Zeichen in einer Zeichenkette ab.

Länge := StrLen(Zkette)

Parameter

Zkette
Die zu messende Zeichenkette.

Rückgabewert

Diese Funktion gibt die Länge der Zeichenkette zurück.

Bemerkungen

Wenn Zkette eine Variable ist, der ClipboardAll zugewiesen wurde, wird StrLen() dessen Gesamtgröße abrufen.

Siehe auch

StringLen, InStr(), SubStr(), Trim(), StringLower, StringUpper, StrPut(), StrGet(), StrReplace(), StrSplit()

Beispiele

Ermittelt und meldet die Anzahl der Zeichen in einer Zeichenkette.

ZketteWert := "Franz jagt im komplett verwahrlosten Taxi quer durch Bayern."
MsgBox % "Die Zeichenkette ist " StrLen(ZketteWert) " Zeichen lang." ; Ergebnis: 60
