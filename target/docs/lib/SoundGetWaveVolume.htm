SoundGetWaveVolume

Ruft die Wave-Wiedergabelautstärke eines Audiogeräts ab.

SoundGetWaveVolume, AusgabeVar , GerätNummer

Parameter

AusgabeVar

Name der Ausgabevariable, in der die abgerufene Lautstärke gespeichert werden soll, was eine Floating-Point-Zahl im Bereich von 0 bis 100 ist. Die Variable wird leer gemacht, wenn beim Abrufen der Lautstärke ein Problem aufgetreten ist. Das Format der Floating-Point-Zahl, wie z.B. die Dezimalstellen, wird durch SetFormat bestimmt.

GerätNummer

Wenn leer oder weggelassen, wird standardmäßig 1 (das erste Audiogerät) verwendet, was normalerweise das Standardgerät des Systems für Aufnahme und Wiedergabe ist. Andernfalls geben Sie eine Zahl größer als 1 an, um ein anderes Audiogerät zu verwenden.

Fehlerbehandlung

[v1.1.04+]: Dieser Befehl ist in der Lage, bei Misserfolg eine Ausnahme auszulösen. Weitere Informationen finden Sie unter Laufzeitfehler.

ErrorLevel wird bei Misserfolg auf 1 und bei Erfolg auf 0 gesetzt.

Bemerkungen

Die aktuelle Wave-Wiedergabelautstärke kann via SoundSetWaveVolume gesetzt werden. Einstellungen wie Master-Lautstärke, Synthesizer, Mikrofon, Stummschaltung, Höhe und Bass können mit SoundSet und SoundGet gesetzt bzw. abgerufen werden.

[v1.1.10+]: In Windows Vista und höher ist dieser Befehl äquivalent zu SoundGet, AusgabeVar, Wave, Volume.

Siehe auch

SoundSetWaveVolume, SoundSet, SoundGet, SoundPlay

Beispiele

Ermittelt und meldet die aktuelle Wave-Wiedergabelautstärke.

SoundGetWaveVolume, AusgabeVar
MsgBox, Die aktuelle Wave-Wiedergabelautstärke ist bei %AusgabeVar%`%.
