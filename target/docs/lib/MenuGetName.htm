MenuGetName() [v1.1.23+]

Ruft den Namen eines Win32-Menüs über dessen Handle ab.

MenüName := MenuGetName(Handle)

Parameter

Handle

Ein Win32-Menü-Handle (HMENU).

Bemerkungen

Es wird nur das aktuelle Handle des Menüs erkannt. Wenn das zum Menü gehörende Win32-Menü zerstört und neu erstellt wird, ändert sich das Handle. Weitere Informationen finden Sie unter Win32-Menüs.
