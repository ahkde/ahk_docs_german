#CommentFlag

Überschreibt das standardmäßig verwendete Kommentarzeichen (Semikolon) mit einer anderen Zeichenkette.

Veraltet: Diese Direktive ist nicht für neue Skripte empfohlen. Verwenden Sie stattdessen die reguläre Kommentarmarkierung (Semikolon).

#CommentFlag NeueZkette

Parameter

NeueZkette

Ein oder mehrere Zeichen, die die neue Kommentarmarkierung definieren. Es können bis zu 15 Zeichen angegeben werden.

Bemerkungen

Wenn diese Direktive im Skript nicht angegeben ist, verhält sie sich so, als wäre sie auf Semikolon (;) gesetzt.

Die Kommentarmarkierung wird verwendet, um dem Skript mitzuteilen, dass der nachfolgende Text ignoriert werden soll (aus Performanzgründen werden Kommentare beim Start eines Skripts nicht in den Speicher geladen).

Eine Kommentarmarkierung neben einem Befehl wird erst als solches angesehen, wenn ihr mindestens ein Leer- oder Tabulatorzeichen vorangestellt ist. Zum Beispiel:

MsgBox, Test1 ; Das ist ein Kommentar.
MsgBox, Test2; Das ist kein Kommentar und wird im Mitteilungsfenster angezeigt.

#CommentFlag kann wie alle anderen Direktiven nicht bedingt ausgeführt werden.

Siehe auch

#EscapeChar

Beispiele

Wechselt zu einer Kommentierung wie bei C++.

#CommentFlag //
