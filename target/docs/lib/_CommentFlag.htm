#CommentFlag

Überschreibt das standardmäßig verwendete Kommentarzeichen (Semikolon) mit einer anderen Zeichenkette.

Veraltet: Diese Direktive ist nicht für neue Skripte empfohlen. Verwenden Sie stattdessen die vordefinierte Kommentarmarkierung (Semikolon).

#CommentFlag NeueZkette

Parameter

NeueZkette

Eine oder mehrere Zeichen, die die neue Kommentarmarkierung definieren. Es können bis zu 15 Zeichen angegeben werden.

Bemerkungen

Standardmäßig ist die Kommentarmarkierung ein Semikolon (;).

Die Kommentarmarkierung wird genutzt, um dem Skript mitzuteilen, dass der darauffolgende Text ignoriert werden soll (um die Leistung zu erhöhen, werden Kommentare nicht in den Speicher geladen, wenn ein Skript gestartet wird).

Eine Kommentarmarkierung neben einem Befehl wird erst als solches angesehen, wenn mindestens ein Leer- oder Tabulatorzeichen davor ist. Zum Beispiel:

MsgBox, Test1 ; Das ist ein Kommentar.
MsgBox, Test2; Das ist kein Kommentar und wird im Mitteilungsfenster angezeigt.

#CommentFlag kann wie alle anderen Direktiven nicht bedingt ausgeführt werden.

Siehe auch

#EscapeChar

Beispiele

Wechselt zu einer Kommentierung wie bei C++.

#CommentFlag //
