FileGetTime

Ermittelt den Zeitstempel einer Datei oder eines Ordners.

FileGetTime, AusgabeVar , DateiName, WelcheZeit

Parameter

AusgabeVar

Name der Ausgabevariable, in der der ermittelte Zeitstempel im YYYYMMDDHH24MISS-Format gespeichert werden soll. Die Zeit ist die lokale Zeit des Betriebssystems, nicht UTC/GMT.

DateiName

Name der Zieldatei oder des Zielordners. Wenn kein absoluter Pfad angegeben ist, wird die Datei oder der Ordner im A_WorkingDir-Verzeichnis vermutet. Lässt man diesen Parameter weg, wird stattdessen die aktuelle Datei von der innersten umschlossenen Datei-Schleife verwendet.

WelcheZeit

Wenn dieser Parameter weggelassen wird oder leer ist, wird standardmäßig M (Zeitpunkt der letzten Änderung) verwendet. Ansonsten können Sie einen der folgenden Buchstaben angeben, um zu bestimmen, welcher Zeitstempel abgerufen werden soll:

  • M = Zeitpunkt der letzten Änderung
  • C = Zeitpunkt der Erstellung
  • A = Zeitpunkt des letzten Zugriffs

Fehlerbehandlung

[v1.1.04+]: Dieser Befehl ist in der Lage, bei Misserfolg eine Ausnahme auszulösen. Weitere Informationen finden Sie unter Laufzeitfehler.

ErrorLevel wird bei Misserfolg auf 1 und bei Erfolg auf 0 gesetzt.

A_LastError wird auf das Ergebnis der Systemfunktion GetLastError() gesetzt.

Bemerkungen

Siehe YYYYMMDDHH24MISS, um mehr über Datum- und Zeitangaben zu erfahren.

Siehe auch

FileSetTime, FormatTime, If Var is [not] Typ, FileGetAttrib, FileSetAttrib, FileGetSize, FileGetVersion, Datei-Schleife, EnvAdd (Datumsberechnung), EnvSub (Datumsdifferenz)

Beispiele

Ermittelt den Zeitpunkt der letzten Änderung und speichert den Wert in AusgabeVar.

FileGetTime, AusgabeVar, C:\Meine Dokumente\test.doc

Ermittelt den Zeitpunkt der Erstellung und speichert den Wert in AusgabeVar.

FileGetTime, AusgabeVar, C:\Meine Dokumente\test.doc, C
